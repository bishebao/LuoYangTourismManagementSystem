<h2>一、项目简介</h2>
基于ssm的学生选课管理系统：角色分为：老师、学生；集成课程管理、上传成绩、选课等功能于一体的系统。
<h2>二、项目技术</h2>
基于java+jsp的洛阳旅游管理系统：集成景点、资讯、留言等功能于一体的系统。
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">网站首页：主导航栏，关键词搜索，天气情况展示，轮播图，当季热门旅游景点，热门资讯
畅游：景点介绍，景点详情展示
资讯：前台资讯内容展示
景点管理：景点信息的增删改查，景点图片上传，景点名称模糊搜索
资讯管理：资讯信息的增删改查，资讯图片上传
留言中心：留言列表查询，留言删除
管理员管理：管理员账号的增删改查
系统管理：基本设置，网站的名称、关键词、描述、底部等信息的设置</div>
</div>
